
Team Task Manager is a simple full-stack web application where users can register, login, and manage tasks. Users can create tasks, update their status, and track progress.

 Features
User Login & Register (JWT Authentication)
Create Tasks
View Tasks
Update Task Status (Pending / In Progress / Done)
Role-based Access (Admin / Member)
MongoDB Database

Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MongoDB
Authentication: JWT, Bcrypt
Deployment: Railway

Project structure 
backend/
  frontend/
    index.html
    dashboard.html
  routes/
  models/
  server.js


  How to Run Locally
cd backend
npm install
node server.js

Then open:
http://localhost:5000
